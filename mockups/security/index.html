Agent profiles

Reusable named bundles of guardrail boundaries — filesystem + syscall + network ACLs, resource limits, allowed tools & LLM budget. Assign to a sandbox UID to make the security boundary legible and repeatable.

Every profile runs unprivileged (L2 LXC) with rootless L3 where enabled. Profiles never grant privilege — they only narrow the default-deny boundary.
