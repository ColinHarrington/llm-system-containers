agent-* = per-agent Linux UID · operator = human operator UID Egress and inspection rules can differ per UID within the same sandbox.
Default-deny egress

No sandbox reaches the internet by default. Outbound is opt-in via an allowlist on egress-net; everything else is dropped.

LLM only via virtual keys

Agents call LiteLLM on svc-net with virtual keys. Real provider credentials live only in the proxy — agents never hold them or reach providers directly.

Inspect what flows

mitmproxy can block/modify HTTP(S) at the app layer; Zeek passively logs every connection at the packet layer for audit.

Kernel-level enforcement

Incus network ACLs and Tetragon (eBPF) decide whether a connection is allowed — per container, per UID, non-bypassable from userspace.
